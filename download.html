常见问题

解答您关于下载和安装的常见问题

如何下载小马办公？

+

点击上方的"立即下载"按钮即可下载最新版本。解压后运行 PonyWork.exe 即可使用。我们也提供便携版，无需安装，复制到任何位置即可运行。如需其他版本，请访问 GitHub Releases 页面。

安装/解压失败怎么办？

+

请确保您的系统满足最低要求。下载后请检查文件完整性，如果问题仍然存在，请尝试关闭杀毒软件后重新下载解压。我们建议解压到纯英文路径目录下。

便携版和普通版有什么区别？

+

下载的是同一个压缩包，解压后即可使用。直接运行 PonyWork.exe，无需安装。如需要可创建桌面快捷方式。适合需要随身携带或不想在系统中留下痕迹的用户。

如何获取更新版本？

+

应用内会提示有新版本可用，您也可以关注我们的GitHub仓库或官方网站获取最新更新。
